# 极限学习 (Extreme Learning)

一个基于 MIT 教学法的 AI 深度学习技能，帮助用户在极短时间内从零掌握任意领域的底层逻辑与深度见解。

## 核心方法论

通过三个递进阶段实现深度掌握：

1. **底层逻辑萃取** — 从专业资料中提炼专家的 5 个核心思维模型
2. **争议地图构建** — 识别领域内 3 个核心争论点及双方论据
3. **极限压力自测** — 10 道高难度逻辑题 + 苏格拉底式追问，直至完全掌握

## 适用场景

- 想快速入门一个陌生领域
- 拥有教材、论文、会议记录等学习材料
- 希望构建系统性的专家思维框架而非死记硬背
- 需要通过高强度练习检验和巩固真正理解
